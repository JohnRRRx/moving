 ## サービス概要  
周辺施設を検索・確認できる、地図ベースの生活環境調査アプリです。  
引越し先の調査や地域探索に活用できます。

  
 ## 開発背景  
 
Google Maps API、React、TypeScriptのキャッチアップを目的に開発しました。  
生活の利便性を「スーパーや駅など周囲にある施設の多さ」で判断する自分の価値観から着想を得ました。
